Board Usage

  1. Product Backlog:
    The Product Backlog is where you'll list all the tasks or features you want to work on in the future. To add an item, click the "Add" button and provide a title and assign it to the responsible person.
  2. Adding Tasks:
    Tasks can be added and tagged with a selection of 8 different categories, each with their own colour. Each can be assigned a user and have their own title.
  3. Sprint:
    The Sprint is where you plan and execute the current work. It consists of the following columns:
    • To-Do Lane:
      In the "To-Do" lane, add tasks that need to be completed in the current sprint. To add a task, click "Add," provide a title, and assign it to the responsible person.
    • Doing Lane:
      When a task is in progress, move it to the "Doing" lane. Drag and drop tasks here when work begins.
    • Done Lane:
      When a task is completed, move it to the "Done" lane. Tasks that are completed in the current sprint go here.
  4. Task Management:
    Click and drag tasks between the columns to reflect their current status. For example, move tasks from "To-Do" to "Doing" when you start working on them.
  5. Task Details:
    You can click on a task to view or edit details. This includes the title and assigned person. Make updates as needed.
  6. Priority:
    You can use the order of tasks in the "To-Do" lane to indicate priority. The top task is the highest priority.
  7. Limit Work-in-Progress (WIP):
    To maintain a smooth workflow, it's a good practice to limit the number of tasks in the "Doing" lane to prevent overloading.
  8. Daily Standup (Optional):
    Consider having a daily standup meeting to discuss progress, obstacles, and task assignments.
  9. Review and Reflect:
    At the end of the sprint, review what was completed and reflect on what can be improved for the next sprint.
  10. Customize as Needed:
    You can customize the board, add or remove lanes, and adapt it to your specific workflow and needs.
